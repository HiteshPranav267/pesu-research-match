# 🎓 PESU ResearchMatch

> Match your skills and interests to the right research professor at PES University — powered by local ML, no internet or API keys required after setup.

Built as a university ML course project. Uses sentence-transformers to semantically match student resumes or interest descriptions to PES faculty across RR and EC campuses.

## 🧠 How It Works

1. **Scrape** — Pulls all faculty names, titles, and departments from staff.pes.edu
2. **Embed** — Converts professor profiles into semantic vectors using MiniLM (runs fully offline)
3. **Match** — Embeds your resume/interests and finds the closest professors using cosine similarity
4. **Display** — React frontend lets you filter by campus & department and shows ranked results

## ⚙️ Setup

### 1. Clone the repo
```bash
git clone https://github.com/yourusername/pesu-research-match.git
cd pesu-research-match
```

### 2. Install dependencies
```bash
pip install requests beautifulsoup4 sentence-transformers fastapi uvicorn numpy scipy
```

### 3. Scrape PES faculty data
```bash
python scraper.py
```
> ⏳ Takes ~5–10 minutes due to pagination across all departments. Saves `professors.json`.

### 4. Generate embeddings
```bash
python embed.py
```
> 📦 First run downloads the MiniLM model (~80MB). Subsequent runs are instant.

### 5. Start the backend
```bash
uvicorn api:app --reload
```
> Runs on http://localhost:8000

### 6. Open the frontend
Open `App.jsx` in a React environment (Vite, CRA, or paste into [stackblitz.com](https://stackblitz.com)).

## 🔍 Features

- 🏫 Filter by **RR Campus** or **EC Campus**
- 📚 Filter by **department** (CS, ECE, Mechanical, etc.)
- 📄 **Upload your resume** (PDF) — text extracted locally in-browser
- ✍️ Or **describe your interests** in plain text
- 🧮 Semantic similarity matching using `all-MiniLM-L6-v2`
- 🔒 Runs **100% locally** — no data leaves your laptop

---

## ⚠️ Limitations

- Individual professor profile pages on staff.pes.edu require a PES login, so matching is based on **publicly available info only** (name, title, department)
- For better accuracy, research areas and publications can be manually added to `professors.json`
- Similarity scores are relative — always visit the professor's profile page for full details
